How it works

Use Avax Scan before the wallet prompt

Avax Scan keeps the first pass simple: search the Avalanche item, read the context, then decide whether the route belongs in your wallet flow. You can compare activity against a public Avalanche block explorer without signing a transaction.

1

Paste what you want to inspect

Enter an Avalanche wallet address, token contract, transaction hash, or DeFi route detail into the checker.

2

Review the live context

The app surfaces activity, token movement, contract interactions, and route details so the decision is not blind.

3

Act from your own wallet

If the route still makes sense, continue from the dashboard and approve only the actions you understand.

What it checks

Avalanche signals that matter before you move

Wallet and token context

Check AVAX, Avalanche C-Chain tokens, contract activity, and transaction history before relying on a route.

DeFi route readiness

Review swap, bridge, staking, and pool details in one Avalanche-focused flow before opening a wallet such as Core.

Non-custodial safety

Avax Scan never needs a seed phrase. Connect with tools like MetaMask only after you have reviewed the live app context.

FAQ

Avax Scan questions

01

What is Avax Scan?

Avax Scan is a non-custodial DeFi checker for Avalanche activity. It helps you review wallets, tokens, transactions, and route context before you decide what to do next.

02

How does Avax Scan work?

Paste an Avalanche address, token, or transaction reference, run the checker, and read the live result in the app. Any wallet action stays under your control.

03

Which chains and assets are supported?

Avax Scan is built around Avalanche and AVAX activity, with supported network details maintained on the supported chains page.

04

How long does a scan take?

The dashboard shows the current result flow in app. Timing depends on the item you enter and the live data needed for that check.

05

What does it cost?

The preview is informational. Any app limits, transaction costs, route costs, or wallet network fees are shown in the app before you act.

06

Is Avax Scan safe to use?

It is designed as a non-custodial checker. You should still verify contracts, read wallet prompts, and reject any action you do not recognize.
